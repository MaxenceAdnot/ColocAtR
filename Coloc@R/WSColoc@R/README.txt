Download the Microsoft Web Platform Installer then fire it up by clicking the downloaded EXE file. 

Then make sure that the following options are added and installed:

Products > Server > IIS: Recommended Configuration
Products > Server > IIS: ASP.NET
Products > Server > IIS: ASP.NET 4.5
Products > Frameworks > .NET 3.5 for Windows 8
Products > Frameworks > .NET 4.5 for Windows 8

+ anything that seems interesting to you

Now press the Options link and where it says, Which Web Server would you like to use? ensure the IIS is selected and not IIS Express
